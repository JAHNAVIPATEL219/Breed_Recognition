# Breed Recognition System

A Deep Learning based Breed Recognition System that identifies cattle and buffalo breeds from images using Convolutional Neural Networks (CNN).

## Features
- Image-based breed prediction
- Flask web application
- Deep Learning model (TensorFlow/Keras)
- Multiple cattle and buffalo breed classification

## Technologies Used
- Python
- TensorFlow
- Keras
- Flask
- HTML/CSS

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
